# SAINT Transformer Explainer

Interactive visual explanations of the SAINT (Self-Attention Supervised Tabular Transformers) architecture used for equity return prediction and portfolio construction.

## Pages

- **[English](./index.html)** — Interactive guide in English
- **[Português](./pt-br/index.html)** — Guia interativo em português

## Topics Covered

1. **Feature Matrix** — How financial indicators are structured for neural processing
2. **Tokenization** — Continuous and discrete feature embedding
3. **Transformer Encoder** — Multi-head self-attention architecture
4. **Self-Supervised Pretraining** — Masked reconstruction and next-step prediction
5. **Prediction Pipeline** — From CLS embeddings to return forecasts
6. **Portfolio Construction** — Signal generation and exposure constraints

## Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks)
- Editorial design with DM Serif Display, DM Mono, and Outfit typography
- Static visualizations for clarity and performance
